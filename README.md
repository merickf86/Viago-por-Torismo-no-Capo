# Viagos por Turismo no Capo

A professional travel website showcasing guided tours in Cape Town, South Africa. Built with **React**, **Vite**, and **Tailwind CSS**, featuring a booking form integrated with **Formspree**, responsive design, and social media links.

---

## Features

- Responsive **Home**, **Tours**, and **Contact** pages  
- Professional **Booking Form** with icons and Formspree integration  
- Floating **WhatsApp button** for instant inquiries  
- Fully **responsive layout** for mobile, tablet, and desktop  
- Footer with **Instagram, Pinterest, Facebook, and WhatsApp** links  
- Consistent **typography, colors, and spacing** across the site  

---

## Technologies Used

- React 18  
- Vite  
- Tailwind CSS  
- Lucide React (icons)  
- Formspree (form submissions)
